# my-first-blog
Simple blog application built with Python Web framework - Django, featuring:
- Showing all posts
- Admin login/logout
- Add new post
- Edit existing post
- Save draft - edit, delete, publish
- authorized add/edit of posts
- comment feature
- PostgreSQL database

Live demo: [here](https://obscure-island-79458.herokuapp.com/)
